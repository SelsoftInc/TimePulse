1. Copy .env.local.example to .env.local:

3. Get Google OAuth credentials:
   
   a) Go to: https://console.cloud.google.com/
   b) Create a new project or select existing
   c) Enable Google+ API
   d) Go to Credentials → Create Credentials → OAuth 2.0 Client ID
   e) Application type: Web application
   f) Add Authorized redirect URI:
      https://goggly-casteless-torri.ngrok-free.dev/api/auth/callback/google
   g) Copy Client ID and Client Secret

4. Update .env.local with your Google credentials:
   
   GOOGLE_CLIENT_ID=your-actual-client-id.apps.googleusercontent.com
   GOOGLE_CLIENT_SECRET=your-actual-client-secret
